MedDRA Browser Portable

本地词典浏览 · 中文界面 · Windows 和 Mac 共用便携包

浏览器不能直接替你运行电脑里的程序，所以便携版分两步打开。

第一步：启动本地服务 Windows 双击 【Windows】第一步：请双击我运行.bat；第一次会在本文件夹下准备本地 Python 运行环境，不需要另外安装 Python。Mac 双击 【Mac】第一步：请双击我运行.command。请保持第一步打开的窗口不要关闭。
第二步：打开浏览器入口 第一运行窗口会在服务启动后自动打开页面。如果没有自动打开，再双击 第二步：双击我开始MedDRA浏览.html。如果服务已经启动，本页会自动进入系统。

进入系统后，如果提示还没有词典，请点“选择词典文件夹”。在文件管理器或 Finder 里选你的 MedDRA 文件夹即可，比如 MedDRA_29_0_Chinese、MedDRA_29_0_English、MedAscii、ascii-290，或它们的上级文件夹。
