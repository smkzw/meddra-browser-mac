# App Review Notes

MedDRA Browser is a local dictionary browser for licensed MedDRA ASCII dictionary files.

Important review notes:

1. The app does not include MedDRA dictionary data.
2. The user must already have a valid MedDRA license and must choose their own local MedDRA ASCII dictionary folder.
3. Dictionary parsing, indexing, searching, hierarchy browsing, SMQ browsing, and Research Bin export run locally on the user's Mac.
4. The app does not upload dictionary files, search terms, Research Bin contents, or usage data to any external server.
5. The app is not a diagnostic, treatment, dosing, or clinical decision support tool. It is a terminology lookup and browsing utility.
6. The app supports Chinese UI with Chinese, English, and bilingual MedDRA display modes when the corresponding local dictionary files are selected by the user.

If the reviewer needs test data, please contact the developer through App Review Resolution Center. The developer cannot distribute MedDRA dictionary files through the app package.

For the current candidate build, the app may require a local MedDRA ASCII folder selected by the user before searches are available.
